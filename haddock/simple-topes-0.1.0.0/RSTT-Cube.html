data Point Source #

A point in a cube.

Constructors

PointUnit

The only point in the unit cube: \(\star : \mathbb{1}\).

PointPair Point Point

A pair of points: \(t : I, s : J \vdash \langle t, s \rangle : I \times J\).

PointSecond Point

PointCon Label [Point]
